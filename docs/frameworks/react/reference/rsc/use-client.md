`'use client'` yalnızca `React Server Components` kullanıyorsanız veya onlarla uyumlu bir kütüphane oluşturuyorsanız gereklidir.




`'use client'` hangi kodun istemci tarafında çalışacağını işaretlemenizi sağlar.
